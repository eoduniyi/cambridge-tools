# Cambridge Tools

Interactive tools for [cambridge-dev.org](https://www.cambridge-dev.org/).

## Projects

### [flyering/](./flyering/)

The Traveling Flyerperson — an interactive exploration of optimal flyer placement and routing in Cambridge, MA. Solves a real-world TSP variant with legal constraints, geographic bounds, and confirmed/estimated placement tracking.

**Stack:** SvelteKit, Leaflet, TypeScript  
**Deploy:** Firebase Hosting
